> Display a collection image.

```php
[webcomic_collection_media]
```

Shortcode content overrides the `size` attribute.

## Attributes

### `string` size
Optional media size.

### `mixed` collection
Optional collection to display an image for.

## Uses
- [get_webcomic_collection_media()](get_webcomic_collection_media())
