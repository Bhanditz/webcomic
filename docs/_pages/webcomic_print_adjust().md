> Display a comic print price adjustment.

```php
webcomic_print_adjust( string $type, mixed $post = null ) : void
```

## Parameters

### `string` $type
The print price adjustment to display.

### `mixed` $post
Optional post to get the price adjustment from.

## Return

`void`
