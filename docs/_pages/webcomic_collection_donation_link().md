> Display a collection donation link.

```php
webcomic_collection_donation_link( string $link = '', mixed $collection = null ) : void
```

## Parameters

### `string` $link
Optional link text, like 'before{{text}}after'.

### `mixed` $collection
Optional collection to display a donation link
for.

## Return

`void`
