> Alter media manager enqueing.

```php
apply_filters( 'webcomic_enqueue_media_manager', bool $enqueue )
```

This filter allows hooks to enqueue the shared media manager functionality.

## Parameters

### `bool` $enqueue
Wether to enqueue the media manager.
