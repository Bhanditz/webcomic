> Display a comic term image.

```php
[webcomic_term_media]
```

Shortcode content overrides the `size` attribute.

## Attributes

### `string` size
Optional media size.

### `mixed` term
Optional term to get an image for.

### `array` args
Optional arguments.

## Uses
- [get_webcomic_term_media()](get_webcomic_term_media())
