> Alter the get_webcomics() arguments.

```php
apply_filters( 'get_webcomics_args', array $args )
```

This filter allows hooks to alter the get_webcomics() arguments before
they're passed to get_posts().

## Parameters

### `array` $args
Optional arguments.
