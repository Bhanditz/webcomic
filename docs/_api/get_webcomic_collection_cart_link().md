> Get a collection cart link.

```php
get_webcomic_collection_cart_link( string $link = '', mixed $collection = null ) : string
```

## Parameters

### `string` $link
Optional link text, like 'before\{\{text}}after'.

### `mixed` $collection
Optional collection to get a cart link for.

## Return

`string`
