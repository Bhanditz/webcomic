> Alter the get_webcomic_prints() arguments.

```php
apply_filters( 'get_webcomic_prints_args', array $args )
```

This filter allows hooks to alter the arguments used by
get_webcomic_prints().

## Parameters

### `array` $args
The arguments to filter.
