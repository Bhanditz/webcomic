> Display a collection cart link.

```php
webcomic_collection_cart_link( string $link = '', mixed $collection = null ) : void
```

## Parameters

### `string` $link
Optional link text, like 'before\{\{text}}after'.

### `mixed` $collection
Optional collection to display a cart link for.

## Return

`void`
