> Display an infinite comic link.

```php
[webcomic_infinite_link]
```

Shortcode content overrides the `link` attribute.

## Attributes

### `string` link
Optional link text, like before\{\{text}}after.

## Uses
- [get_webcomic_infinite_link()](get_webcomic_infinite_link())
