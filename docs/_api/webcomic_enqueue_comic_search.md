> Alter comic search enqueing.

```php
apply_filters( 'webcomic_enqueue_comic_search', bool $enqueue )
```

This filter allows hooks to enqueue the shared comic search functionality.

## Parameters

### `bool` $enqueue
Wether to enqueue the comic search.
