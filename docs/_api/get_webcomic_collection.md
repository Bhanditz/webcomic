> Alter the collection.

```php
apply_filters( 'get_webcomic_collection', string $collection, mixed $type )
```

This filter allows hooks to alter the returned collection.

## Parameters

### `string` $collection
The collection ID.

### `mixed` $type
Optional type of collection to get.
