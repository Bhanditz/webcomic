> Display a comic link.

```php
webcomic_link( string $link = '%title', mixed $post = null, array $args = [] ) : void
```

## Parameters

### `string` $link
Optional link text, like before\{\{text}}after.

### `mixed` $post
Optional post to get a link for.

### `array` $args
Optional arguments.

## Return

`void`
