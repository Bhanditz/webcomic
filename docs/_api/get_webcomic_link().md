> Get a comic link.

```php
get_webcomic_link( string $link = '%title', mixed $post = null, array $args = [] ) : string
```

## Parameters

### `string` $link
Optional link text, like before\{\{text}}after.

### `mixed` $post
Optional post to get a link for.

### `array` $args
Optional arguments.

## Return

`string`
