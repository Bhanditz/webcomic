> Get a term comic count.

```php
get_webcomic_storyline_count( mixed $term = null, array $args = [] ) : int
```

## Parameters

### `mixed` $term
Optional term to get a post count for.

### `array` $args
Optional arguments.

## Return

`int`

## Uses
- [get_webcomic_term_count()](get_webcomic_term_count())  
The type argument is always set to
`storyline`.
