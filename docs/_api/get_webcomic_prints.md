> Alter the prints.

```php
apply_filters( 'get_webcomic_prints', array $prints, array $args )
```

This filter allows hooks to alter the list of prints before standard
arguments are applied to the prints array.

## Parameters

### `array` $prints
The prints array.

### `array` $args
Optional arguments.
