> Get a comic storyline.

```php
get_webcomic_storyline( mixed $term = null, array $args = [] ) : mixed
```

## Parameters

### `mixed` $term
Optional term to get.

### `array` $args
Optional arguments.

## Return

`mixed`

## Uses
- [get_webcomic_term()](get_webcomic_term())  
The type argument is always set to `storyline`.
